# 弹出页面 / 筛选页面 / 预览页面

![](/assets/filter-page.png)

扩展资源列表页面(popup页) 点击 `弹出` 按钮 打开筛选页面。

::: info
单个页面最大支持同时预览128个文件(还得看你的内存大小...) 大于128个文件分页处理。
:::

## 按钮说明

**标题：** 显示抓取文件时，当时的网页标题。

**在线合并：** 选择一个视频和一个音频，进行音频视频合并。

**复制：** 复制选中文件的地址。

**删除：** 临时删除文件，刷新页面还原文件列表。

**清空：** 清空文件列表，无法恢复。

**删除无法预览的文件:** 删掉预览失败的媒体文件。

**debug：** 你不需要它。



::: warning
关闭网站后猫抓会清空属于当前网站的数据，使用本功能预览文件时，请勿关闭被抓取的网站。
:::
