## 兼容性

扩展至少需要 Chromium 93 及以上内核版本的浏览器。

捕获脚本/脚本注入 功能需要 102 及以上版本，否则不显示相关按钮。屏幕录制 需要 104 及以上版本。

Firefox 需要 93 及以上。

低于93版本的浏览器，可以使用1.0.16版本猫抓。

## **GitHub Releases 版本说明** {#version}

**cat-catch-\*.\*.\*-chrome.crx** Chrome 扩展商店版，无法访问 Chrome 扩展网站的用户可以安装，文件拖动到扩展页面安装，需提前在扩展管理页面打开开发者模式，安装后扩展能自动更新版本。

**cat-catch-\***_**.\***_**.\*-edge.crx** Edge 扩展商店版，文件拖动到扩展页面安装，需提前在扩展管理页面打开开发者模式，安装后扩展能自动更新版本。

**cat-catch-\*.\*.\*.crx** 通用版本，支持所有 Chromium 93 及以上内核浏览器，扩展不能自动更新版本。

**cat-catch-\*.\*.\*-firefox.xpi** Firefox 浏览器版，文件拖动到扩展页面安装，可以自动更新。

::: info
使用Chromium内核浏览器不能直接下载crx文件，需右键另存为。
:::
