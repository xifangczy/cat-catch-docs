# Popup Page / Filter Page / Preview Page

![](/assets/filter-page.png)

The extension resource list page (popup page) — click the `Popup` button to open the filter page.

::: info
A single page supports previewing up to 128 files at the same time (also depends on your memory size...). More than 128 files will be paginated.
:::

## Button Descriptions

**Title:** Displays the web page title at the time the file was captured.

**Online Merge:** Select one video and one audio to merge audio and video.

**Copy:** Copy the address of the selected file.

**Delete:** Temporarily delete a file; refreshing the page restores the file list.

**Clear:** Clear the file list; this cannot be undone.

**Delete files that cannot be previewed:** Remove media files that failed to preview.

**debug:** You don't need this.

::: warning
When using this feature to preview files, do not close the website being captured.
:::
