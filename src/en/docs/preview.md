# Popup Page / Filter Page / Preview Page

![](/assets/filter-page.png)

On the extension resource list page (popup), click the `Popup` button to open the filter page.

::: info
A single page supports previewing up to 128 files simultaneously (depending on your memory). More than 128 files are paginated.
:::

## Button Descriptions

**Title:** Displays the webpage title at the time of capture.

**Online Merge:** Select one video and one audio to merge them.

**Copy:** Copy the URL of selected files.

**Delete:** Temporarily delete files. Refresh the page to restore.

**Clear All:** Clear the file list. Cannot be undone.

**Delete Unpreviewable Files:** Remove media files that failed to preview.

**Debug:** You don't need this.

::: warning
Do not close the captured website while using this feature to preview files.
:::
