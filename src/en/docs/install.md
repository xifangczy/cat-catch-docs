## Compatibility

The extension requires Chromium 93 or later.

The Capture Script / Script Injection feature requires version 102 or later, otherwise the related buttons will not be displayed. Screen Recording requires version 104 or later.

Firefox requires version 93 or later.

For browsers older than version 93, you can use Cat Catch version 1.0.16.

## **GitHub Releases Version Notes** {#version}

**cat-catch-\*.\*.\*-chrome.crx** Chrome Web Store version. Users who cannot access the Chrome Web Store can install it by dragging the file to the extensions page. Developer mode must be enabled in the extensions management page first. The extension can auto-update after installation.

**cat-catch-\*.\*.\*-edge.crx** Edge Add-ons Store version. Install by dragging the file to the extensions page. Developer mode must be enabled in the extensions management page first. The extension can auto-update after installation.

**cat-catch-\*.\*.\*.crx** Universal version. Supports all Chromium 93+ browsers. The extension cannot auto-update.

**cat-catch-\*.\*.\*-firefox.xpi** Firefox browser version. Install by dragging the file to the extensions page. Can auto-update.

::: info
When using Chromium-based browsers, you cannot directly download crx files. Right-click and select "Save link as...".
:::
