## Compatibility

The extension requires a Chromium-based browser version 93 or higher.

The capture script / script injection feature requires version 102 or higher, otherwise the related buttons will not appear. Screen recording requires version 104 or higher.

Firefox requires version 93 or higher.

For browsers below version 93, you can use cat-catch version 1.0.16.

## **GitHub Releases Version Guide** {#version}

**cat-catch-\*.\*.\*-chrome.crx** — Chrome Web Store version. For users who cannot access the Chrome Web Store. Drag the file onto the extensions page to install. Enable Developer mode in the extensions management page first. The extension will auto-update after installation.

**cat-catch-\*.\*.\*-edge.crx** — Edge Add-on Store version. Drag the file onto the extensions page to install. Enable Developer mode in the extensions management page first. The extension will auto-update after installation.

**cat-catch-\*.\*.\*.crx** — Universal version. Supports all Chromium 93+ browsers. The extension will NOT auto-update.

**cat-catch-\*.\*.\*-firefox.xpi** — Firefox version. Drag the file onto the add-ons page to install. Supports auto-update.

::: info
Chromium-based browsers cannot download .crx files directly. Right-click and select "Save link as..." instead.
:::
