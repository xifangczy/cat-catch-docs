# Popup Page

![Popup Page](/assets/popup.png)

![Special Resources](/assets/image.png) If a resource has this icon, it means the resource comes from deep search, regex matching, Tampermonkey scripts, or other special sources.

## Merge Download

Many websites separate video and audio. When you select two resources (one video and one audio), `Merge Download` becomes available. The cat-catch downloader downloads the audio and video separately, then merges them using online ffmpeg.

## Popup

Jumps to a more comprehensive filter page. See [Filter / Preview Page](preview) for details.

## More Features - Mobile Simulation

Click to refresh the current page with a mobile User-Agent, making the website display its mobile version. Many resources that cannot be downloaded or sniffed on the desktop version can be accessed on the mobile version.

## More Features - Auto Download

Once enabled, resources are downloaded immediately as they are sniffed. This is useful for sites where you can't find the M3U8 but TS segments keep appearing. You can download while playing and merge them later with a third-party tool.
